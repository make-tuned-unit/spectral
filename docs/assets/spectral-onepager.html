The economics

Memory that’s free by construction

Not “cheap” — structurally free. Recall and recognition make zero model calls, so you can assert the memory layer adds no per-query cost, not just hope it stays small.

Where the name comes from

Recognition: “have I seen this?”

Most memory only recalls. Spectral also recognizes — scoring a new stimulus as familiar or novel, deterministically, with the exact features behind the verdict.

Its landmarks are a stimulus’s statistically salient features — the text analog of spectral peaks above the noise floor. That’s the name.

  • Landmark fingerprinting — IDF-salient features (Shazam-style)
  • Winnowed k-grams — the MOSS verbatim-run guarantee
  • REM + MINERVA-2 — cognitive-psych familiarity scoring
